# Privacy Policy Website

This is a standalone privacy policy website for QuitBet AI that can be hosted anywhere.

## Quick Setup Options

### Option 1: GitHub Pages (FREE & EASIEST) ⭐ Recommended

1. **Create a new GitHub repository:**
   - Go to https://github.com/new
   - Name it: `quitbet-ai-privacy-policy`
   - Make it **Public**
   - Click "Create repository"

2. **Upload the files:**
   ```powershell
   cd privacy-policy-website
   git init
   git add index.html
   git commit -m "Add privacy policy"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/quitbet-ai-privacy-policy.git
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to your repository on GitHub
   - Click **Settings** → **Pages**
   - Under "Source", select **main branch** and **/ (root)**
   - Click **Save**

4. **Your privacy policy will be live at:**
   ```
   https://YOUR_USERNAME.github.io/quitbet-ai-privacy-policy/
   ```

### Option 2: Netlify (FREE & EASY)

1. Go to https://www.netlify.com
2. Sign up (free)
3. Drag and drop the `privacy-policy-website` folder onto Netlify
4. Your site will be live instantly at: `https://random-name.netlify.app`
5. You can add a custom domain later if you want

### Option 3: Vercel (FREE)

1. Go to https://vercel.com
2. Sign up (free)
3. Click "Add New Project"
4. Import the `privacy-policy-website` folder
5. Deploy - it will be live instantly

### Option 4: Any Web Hosting

Just upload the `index.html` file to any web hosting service.

## Before Publishing

**IMPORTANT:** Update the email address!

1. Open `index.html`
2. Search for: `YOUR_EMAIL_HERE@example.com`
3. Replace with your actual email address (e.g., `yourname@gmail.com`)
4. Save and deploy

## For App Store Submission

Use your privacy policy URL in App Store Connect:
- **Privacy Policy URL**: `https://YOUR_USERNAME.github.io/quitbet-ai-privacy-policy/` (or your Netlify/Vercel URL)
